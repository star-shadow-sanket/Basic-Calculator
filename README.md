# 🧮 Modern Calculator (HTML, CSS, JavaScript)

A simple **Modern Calculator Web Application** built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations with a clean and modern user interface.

---

## 🚀 Features

* Modern and responsive calculator UI
* Supports basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Clear button to reset the display
* Interactive button animations
* Error handling for invalid expressions
* Single-page application

---

## 🛠 Technologies Used

* **HTML5** – Structure of the calculator
* **CSS3** – Styling and layout design
* **JavaScript** – Calculator logic and operations

---

## 📂 Project Structure

```text
calculator-project/
│
└── Basic Calculator.html
```

The project is implemented in a **single HTML file containing embedded CSS and JavaScript**.

---

## ▶️ How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **Basic Calculator.html**.
4. The calculator will open in your browser.

---

## 📸 Application Overview

The calculator interface includes:

* A **display screen** to show numbers and results
* **Number buttons (0–9)**
* **Operator buttons (+, −, ×, ÷)**
* **Clear button**
* **Equal button (=)** to calculate results
